# 🪙 FinSense AI: Intelligent Wealth Companion

FinSense is a next-generation personal finance coach that goes beyond static charts. Using LLMs and real-time data, it analyzes spending patterns to provide proactive behavioral nudges and automated budgeting.

## ✨ Key Features
- **AI Behavioral Insights:** "You've spent 15% more on coffee this week. Switching to home-brew could save you $40/month."
- **Predictive Forecasting:** Forecasts your end-of-month balance based on recurring subscriptions and historical trends.
- **Privacy-First:** AES-256 encryption for all linked financial data.
- **Smart Categorization:** Automatically sorts transactions using NLP.

## 🛠️ Tech Stack
- **Frontend:** Next.js 14, Tailwind CSS, Framer Motion (Animations).
- **Backend:** FastAPI, PostgreSQL.
- **AI Engine:** Gemini API / OpenAI API for financial advisory logic.
- **Auth:** Clerk / NextAuth.js.

## 🚀 Getting Started
1. Clone the repo: `git clone https://github.com/username/finsense-ai.git`
2. Install dependencies: `npm install`
3. Set up your `.env` with your AI API keys.
4. Run locally: `npm run dev`
